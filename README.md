# Cartoon Artist Classification

Classification of cartoon drawings by artist using 
deep learning (CNN) and traditional ML (SVM + PCA/PLS).

## Approaches
- **Deep Learning**: Transfer learning with ResNet/EfficientNet
- **Classical ML**: Feature extraction + PCA/PLS + SVM

## Dataset
- X classes (artists), ~Y images/class
- Augmented via Roboflow
